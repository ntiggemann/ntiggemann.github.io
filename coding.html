Plotting tropical curves

You can find a python script for plotting tropical curves defined by a tropical polynomial here. The script also plots the dual subdivision.
It uses the python packages numpy, matplotlib and tkinter (and math and re).
How to use this scipt: First install python and all the required packages listed above. Running this script will then open a window with several options. The input of the polynomial has to be of the following form: (cf singular). Hitting the plot button will generate two figures, one of the curve, one of its figures and, print all the things you specified in the settings.
